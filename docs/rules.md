# VocabNote - Coding Rules

1. **Keep it Simple:** Prefer the easiest working solution over complex code.
2. **Beginner Friendly:** All code must be heavily commented so a beginner can read and understand it.
3. **Notebook First, AI Second:** The app must open and work perfectly for saved words even if there is no internet or the API key is missing.
4. **Always-Visible Export:** The Export button must NEVER be hidden inside a menu. It lives on the main screen.
5. **Modular Code:** Keep UI code (how it looks) separate from Database code (how it saves) and API code (how it fetches).
